## Install Package

Add the [PowerSync Capacitor NPM package](https://www.npmjs.com/package/@powersync/capacitor) to your project:

<Tabs>
  <Tab title="npm">
    ```bash
    npm install @powersync/capacitor
    ```
  </Tab>

  <Tab title="yarn">
    ```bash
    yarn add @powersync/capacitor
    ```
  </Tab>

  <Tab title="pnpm">
    ```bash
    pnpm install @powersync/capacitor
    ```
  </Tab>
</Tabs>

<Info>
  This package uses [`@powersync/web`](https://www.npmjs.com/package/@powersync/web) as a peer dependency. For additional `@powersync/web` configuration and instructions see the [Web SDK README](/client-sdk-references/javascript-web).
</Info>

## Install Peer Dependencies

You must also install the following peer dependencies:

<Tabs>
  <Tab title="npm">
    ```bash
    npm install @capacitor-community/sqlite @powersync/web @journeyapps/wa-sqlite
    ```
  </Tab>

  <Tab title="yarn">
    ```bash
    yarn add @capacitor-community/sqlite @powersync/web @journeyapps/wa-sqlite
    ```
  </Tab>

  <Tab title="pnpm">
    ```bash
    pnpm install @capacitor-community/sqlite @powersync/web @journeyapps/wa-sqlite
    ```
  </Tab>
</Tabs>

See the [Capacitor Community SQLite repository](https://github.com/capacitor-community/sqlite) for additional instructions.

## Sync Capacitor Plugins

After installing, sync your Capacitor project:

```bash
npx cap sync
```

<Info>
  By default, this SDK connects to a PowerSync instance via WebSocket (from `@powersync/capacitor@1.6.0`) or HTTP streaming (before `@powersync/capacitor@1.6.0`). See [Developer Notes](/client-sdk-references/capacitor#developer-notes) for more details on connection methods.
</Info>
