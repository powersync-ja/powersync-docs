PowerSync automatically syncs your backend database to a local SQLite database on the device. Your app reads and writes directly to this database, so it responds instantly and stays fully functional without a network connection.

PowerSync is made up of the PowerSync Service and a set of client SDKs. The PowerSync Service replicates data from your backend database (Postgres, MongoDB, MySQL, or SQL Server), streams real-time updates to clients through the PowerSync SDK, and gives you full control over which data each user receives.
