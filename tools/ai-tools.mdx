# PowerSync and AI Coding Tools

A growing collection of resources to help you work with PowerSync using AI-powered tools like Cursor, Claude, Windsurf, and ChatGPT. These resources give AI assistants the context they need to help you implement PowerSync features faster.

## PowerSync Agent Skills

PowerSync Agent Skills provides AI agents with the PowerSync-specific context and capabilities needed to act on tasks in your codebase. This includes knowledge of PowerSync APIs, patterns, and best practices.

To get started, run the following command in your terminal and follow the prompts:

```bash
npx skills add powersync-ja/agent-skills
```

## AI-Accessible Documentation

### Contextual Menu

<Frame caption="Context menu on a docs page with options to copy content, view as Markdown, open in an AI platform, or connect to an MCP-compatible editor.">
  <img src="/images/tools/context-menu-llm.png" alt="Context menu with AI-related options" />
</Frame>

Every page in our documentation includes a contextual menu with AI-friendly options:

- **Copy page** — exports the current page as Markdown
- **View as Markdown** — opens the page in Markdown format
- **Open in ChatGPT / Claude** — starts a conversation in your preferred AI platform with the page content pre-loaded
- **Connect to Cursor / VS Code** — copies the MCP server URL or install command for use in your editor

You can also append `.md` to any documentation URL to get the Markdown version directly:

```
https://docs.powersync.com/intro/setup-guide.md
```

Or press **Ctrl/Cmd+C** on any page to copy it as Markdown.

### Full Documentation Text

To give AI assistants broad context about PowerSync, we provide two plain-text versions of our documentation:

- **[llms-full.txt](https://docs.powersync.com/llms-full.txt)** — the entire documentation site in a single file, ideal for loading complete context into an AI assistant
- **[llms.txt](https://docs.powersync.com/llms.txt)** — a page outline listing all documentation pages, useful for helping AI assistants index and navigate our docs
