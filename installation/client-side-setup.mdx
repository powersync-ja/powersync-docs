## Supported Platforms

PowerSync supports multiple app frameworks. Select your platform below for specific installation instructions:

<AccordionGroup>
  <Accordion title="Flutter" icon="flutter">
    <Info>
      The PowerSync Flutter SDK is available through pub.dev
    </Info>

    Install the package using:
    ```bash
    flutter pub add powersync
    ```

    For detailed implementation guides and API reference:
    <Card title="Flutter SDK Documentation" icon="flutter" href="/client-sdk-references/flutter" horizontal />
  </Accordion>

  <Accordion title="React Native & Expo" icon="react">
    <Warning>
      **Important**: PowerSync is not compatible with Expo Go. You must use Expo Dev Builds due to native plugin dependencies.
    </Warning>

    Install the package using your preferred package manager:

    <Tabs>
      <Tab title="npm">
        ```bash
        npm npx expo install @powersync/react-native
        ```
      </Tab>

      <Tab title="yarn">
        ```bash
        yarn expo add @powersync/react-native
        ```
      </Tab>

      <Tab title="pnpm">
        ```bash
        pnpm dlx expo install @powersync/react-native
        ```
      </Tab>
    </Tabs>

    <Note>
      **SDK Version Notes**:
      - From v1.9.0: Most polyfills are bundled (except Async Iterator)
      - From v1.11.0: WebSocket is the default connection method
    </Note>

    For complete setup instructions and requirements:
    <Card title="React Native & Expo Guide" icon="react" href="/client-sdk-references/react-native-and-expo" horizontal />
  </Accordion>

  <Accordion title="JavaScript/Web" icon="js">
    Install the package using your preferred package manager:

    <Tabs>
      <Tab title="npm">
        ```bash
        npm install @powersync/web
        ```
      </Tab>

      <Tab title="yarn">
        ```bash
        yarn add @powersync/web
        ```
      </Tab>

      <Tab title="pnpm">
        ```bash
        pnpm install @powersync/web
        ```
      </Tab>
    </Tabs>

    <Note>
      **SDK Version Notes**:
      - From v1.4.0: Polyfills are bundled in the SDK
      - From v1.6.0: WebSocket is the default connection method
    </Note>

    For complete documentation:
    <Card title="JavaScript Web Guide" icon="js" href="/client-sdk-references/javascript-web" horizontal />
  </Accordion>

  <Accordion title="Kotlin Multiplatform" icon="flag">
    <Info>
      Configure your build.gradle.kts to include PowerSync
    </Info>

    Add the following to your build configuration:
    ```gradle
    kotlin {
        sourceSets {
            commonMain.dependencies {
                api("com.powersync:core:$powersyncVersion")
                // Optional: Supabase Connector
                implementation("com.powersync:connectors:$powersyncVersion")
            }
        }
    }
    ```

    For iOS targets using Cocoapods:
    ```gradle
    cocoapods {
        pod("powersync-sqlite-core") {
            linkOnly = true
        }

        framework {
            isStatic = true
            export("com.powersync:core")
        }
    }
    ```

    For complete setup instructions:
    <Card title="Kotlin Multiplatform Guide (Beta)" icon="flag" href="/client-sdk-references/kotlin-multiplatform" horizontal />
  </Accordion>

  <Accordion title="Swift" icon="swift">
    <Tabs>
      <Tab title="Package.swift">
        ```swift
        let package = Package(
            dependencies: [
                .package(
                    url: "https://github.com/powersync-ja/powersync-swift",
                    exact: "<version>"
                ),
            ],
            targets: [
                .target(
                    name: "YourTargetName",
                    dependencies: [
                        .product(
                            name: "PowerSync",
                            package: "powersync-swift"
                        )
                    ]
                )
            ]
        )
        ```
      </Tab>
      <Tab title="Xcode">
        1. Add package dependency in Xcode
        2. Use URL: `https://github.com/powersync-ja/powersync-swift.git`
        3. Specify exact version (e.g., `1.0.0-Beta.x`)
